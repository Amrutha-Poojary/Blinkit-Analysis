# 🛒 Blinkit Sales Analysis Dashboard

## 📌 Project Overview
This project presents an interactive Power BI Dashboard that analyzes Blinkit's sales performance across Tier 3 outlet locations. The dashboard provides insights into sales trends, outlet performance, customer preferences, and product categories, helping businesses make data-driven decisions.

---

## 📊 Dashboard Summary

| Metric | Value |
|---------|---------|
| 💰 Total Sales | $472.13K |
| 🛒 Total Items Sold | 3,350 |
| 💵 Average Sales per Item | $141 |
| ⭐ Average Customer Rating | 3.9 / 5 |

---

## 🔍 Key Insights

### 🏪 Outlet Performance
- Supermarket Type1 generated the highest sales with **$131.81K**.
- High-sized outlets contributed the most revenue, generating **$299.37K** in sales.
- Tier 3 outlets demonstrated strong sales performance across different outlet categories.

### 🛍️ Product Analysis
- Sales are evenly distributed between **Low Fat** and **Regular** fat content products.
- All product categories maintain an average customer rating of approximately **4 stars**.
- Average sales remain consistent across outlets at around **$141 per item**.

### 📈 Business Insights
- Large outlet sizes significantly impact overall revenue generation.
- Customer preferences show balanced demand for both Low Fat and Regular products.
- Consistent product ratings indicate stable customer satisfaction across categories.
